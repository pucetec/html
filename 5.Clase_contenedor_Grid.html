Clase: Contenedor CSS Grid

Maquetación bidimensional: filas, columnas y áreas con CSS Grid.

Objetivo

Comprender la sintaxis básica de CSS Grid y crear un layout simple usando filas, columnas, grid-template-areas y filas implícitas con grid-auto-rows.

1) ¿Qué es CSS Grid?

CSS Grid es un sistema de maquetación bidimensional que organiza el contenido en filas y columnas. A diferencia de Flexbox (principalmente un eje), Grid trabaja ambos ejes al mismo tiempo.

Activa el grid en el contenedor con display: grid;. Los hijos directos pasan a ser grid items.

2) Propiedades básicas

Propiedad Función Ejemplo
display: grid Convierte el contenedor en grid container. display: grid;
grid-template-columns Define columnas (cantidad y tamaño). repeat(3, 1fr)
grid-template-rows Define filas explícitas (cantidad y tamaño). 60px 1fr 60px
grid-auto-rows Tamaño de las filas implícitas (las que el grid crea cuando hay más ítems de los definidos en grid-template-rows). 100px, minmax(80px, auto)
gap Espacio entre filas y columnas. gap: 16px;
justify-items / align-items Alineación horizontal y vertical dentro de cada celda. start, center, end
grid-template-areas Layout con nombres de áreas (header, sidebar, etc.). "header header"

Valores frecuentes en columnas

  • grid-template-columns: 200px 1fr 2fr; → tres columnas con ancho fijo y fracciones flexibles.
  • grid-template-columns: repeat(3, 1fr); → tres columnas iguales.

CSS del ejemplo

.grid-ejemplo {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 12px;
}
.grid-item {
  padding: 18px;
  text-align: center;
  border-radius: 10px;
}

4) Ejemplo con grid-template-areas

Layout con encabezado, barra lateral, contenido y pie usando nombres de áreas en el contenedor y grid-area en cada hijo.

Header
Sidebar
Contenido principal
Footer

CSS del ejemplo

.layout-ejemplo {
  display: grid;
  grid-template-columns: 200px 1fr;
  grid-template-rows: 60px 1fr 60px;
  grid-template-areas:
    "header header"
    "sidebar content"
    "footer footer";
  gap: 10px;
}
.layout-header { grid-area: header; }
.layout-sidebar { grid-area: sidebar; }
.layout-content { grid-area: content; }
.layout-footer { grid-area: footer; }

5) Ejemplo con grid-auto-rows

Solo definimos una fila explícita de 56px con grid-template-rows. Los ítems que no caben en esa fila generan filas implícitas; su altura la controla grid-auto-rows: 100px.

La primera fila (ítems 1 y 2) mide 56px. Las filas siguientes (ítems 3 en adelante) miden 100px cada una.

1 · fila explícita
2 · fila explícita
3 · fila implícita
4 · fila implícita
5 · fila implícita

CSS del ejemplo

.grid-auto-ejemplo {
  display: grid;
  grid-template-columns: repeat(2, 1fr);
  grid-template-rows: 56px;      /* solo la 1.ª fila es explícita */
  grid-auto-rows: 100px;         /* altura de las filas que se crean solas */
  gap: 10px;
}
