# HTML and Git - Web Development Course

This repository contains HTML and Git course materials for the Web Development course at PUCETEC.

## 📚 About This Project

This is an educational repository designed to help students learn HTML fundamentals, the basics of how websites work,
and the Git workflow used to submit class work. It includes HTML-based class materials, examples, quick guides, and mini
challenges.

## 📁 Repository Structure


## 📖 Available Classes

### Class 1: Introduction to HTML
**File:** `1.Clase_introduccion_html.html`

Topics covered:
- How a website works
- HTML history and purpose
- Basic HTML document structure
- Main HTML components
- Primary input fields
- Secondary HTML components
- CSS style sheets
- HTTP and common methods: GET, POST, PUT, DELETE and HEAD

### Class 3: Ubuntu and HTML Semantics
**File:** `3.Clase_ubuntu_semantica.html`

Topics covered:
- Creating a user in Ubuntu
- Assigning administrator groups
- HTML semantic structure
- Semantic forms with labels, inputs, radio buttons, checkboxes and selects

### Class 4: DIV, Display and Flexbox
**File:** `4.Clase_div_display_flexbox.html`

Topics covered:
- Using `div` as a generic container
- Understanding `display: block`, `inline`, `inline-block` and `flex`
- Practicing Flexbox direction, alignment, spacing and gaps
- Building responsive layouts with Flexbox

## 🧰 Git Materials

### Quick Guide: Git + GitLab
**File:** `2.Guia_Rapida_Git.html`

Topics covered:
- Creating and registering an SSH key
- Cloning a GitLab repository
- Creating a working branch
- Adding files and creating commits
- Configuring Git user name and email
- Pushing a branch to GitLab

## 💻 How to Use the Class Materials

1. Open the HTML files directly in your web browser.
2. Follow the examples and read the explanations.
3. Use the Git guide when submitting work through GitLab.
4. Complete the mini challenge at the end of each material.
