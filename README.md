# HTML - Web Development Course

This repository contains HTML course materials for the Web Development course at PUCETEC.

## 📚 About This Project

This is an educational repository designed to help students learn HTML fundamentals and the basics of how websites work.
It includes HTML-based class materials, examples, and mini challenges.

## 📁 Repository Structure

```
html/
├── README.md
├── 1.Clase_introduccion_html.html
├── assets/
│   └── style.css
└── LICENSE
```

## 📖 Available Classes

### Class 1: Introduction to HTML
**File:** `1.Clase_introduccion_html.html`

Topics covered:
- How a website works
- HTML history and purpose
- Basic HTML document structure
- Main HTML components
- Primary input fields
- Secondary HTML components
- CSS style sheets
- HTTP and common methods: GET, POST, PUT, DELETE and HEAD

## 💻 How to Use the Class Materials

1. Open the HTML files directly in your web browser.
2. Follow the examples and read the explanations.
3. Complete the mini challenge at the end of the class.
