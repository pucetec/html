Clase: <div>, display y Flexbox

Organización visual del contenido: del flujo normal a los layouts con flexbox.

Objetivo

Comprender el comportamiento de los elementos block, inline, inline-block y aplicar display:flex para crear distribuciones responsivas y ordenadas.

1) ¿Qué es un <div>?

Es una caja genérica de bloque (sin significado semántico) usada para agrupar contenido y aplicar estilos. Úsala con criterio: si existe una etiqueta semántica que describa mejor el contenido, prefiérela. 

<div class="tarjeta">
  <h2>Título</h2>
  <p>Texto dentro de un div.</p>
</div>

2) Propiedad display

Valor Comportamiento Ejemplos de uso
block Ocupa todo el ancho; inicia en línea nueva. Se pueden definir ancho/alto. <div>, <p>, <h1>
inline Ocupa solo el ancho del contenido; no inicia en línea nueva. No toma ancho/alto. <span>, <a>
inline-block Como inline, pero admite ancho/alto y márgenes verticales. Badges, chips, botones personalizados
flex Convierte el elemento en contenedor flexible para organizar ítems en fila o columna. Navbars, tarjetas, grids simples

3) Flexbox práctico

Flexbox distribuye elementos en una dimensión (fila o columna), facilita alineación y espacio entre ellos.
